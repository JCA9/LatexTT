\chapter{Trabajo Futuro}

Debido a que ya se ha completado la fase de diseño del sistema, el frontend de la plataforma WEB y
con las primeras pruebas del OCR, el siguiente paso es avanzar hacia la implementación completa del 
backend junto con el desarrollo de la app móvil y la integración de ambos componentes. Además, 
se busca optimizar el rendimiento del sistema y mejorar la experiencia del usuario mediante pruebas y 
retroalimentación continua.

Nos enfocaremos en la incorporación de estos componentes para lograr un sistema robusto y eficiente que 
cumpla con los objetivos planteados. Además, se considerará la escalabilidad del sistema para futuras
actualizaciones y mejoras, asegurando que la plataforma pueda adaptarse a las necesidades cambiantes 
de los usuarios.

Planeamos tener el sistema listo a mas tardar a finales de mayo de 2026, para buscar retroalimentación antes
de la entrega final del proyecto en junio de 2026.

Finalmente, se planificará una fase de pruebas exhaustivas para garantizar la calidad y fiabilidad del
sistema antes de buscar un lanzamiento oficial.
