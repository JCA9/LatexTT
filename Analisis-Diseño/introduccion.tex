\chapter{Introducción}

\section{Propósito del Documento}

Este documento presenta el análisis y diseño del sistema de gestión financiera personal desarrollado como Trabajo Terminal. El sistema está orientado a jóvenes adultos que buscan establecer hábitos financieros saludables mediante el control automatizado de gastos y la aplicación de metodologías probadas de presupuestación.

\section{Alcance}

El análisis comprende la especificación completa de casos de uso, reglas de negocio, arquitectura del sistema y diseño de interfaces. Se incluye la integración de tecnologías de reconocimiento óptico de caracteres (OCR) para la captura automatizada de tickets de compra.

\section{Metodología Financiera}

El sistema implementa los principios del método de Dave Ramsey, enfocándose en:
\begin{itemize}
    \item Presupuestación por sobres (máximo 10 categorías)
    \item Control estricto de gastos
    \item Planificación de pago de deudas
    \item Educación financiera práctica
\end{itemize}

\section{Audiencia}

Este documento está dirigido a:
\begin{itemize}
    \item Desarrolladores del sistema
    \item Evaluadores académicos
    \item Usuarios finales interesados en el funcionamiento interno
    \item Futuros mantenedores del código
\end{itemize}
